# e-plantShopping
## Project Description
This is an online plant shopping app. The project allows users to browse different types of plants, add them to their cart, and easily purchase their green pets from the comfort of their home.

## Features
* Plant catalogue view
* Categorisation (indoor plants, outdoor plants, succulents)
* Shopping cart management system
* Responsive design

## Technologies
* HTML5
* CSS3
* JavaScript
